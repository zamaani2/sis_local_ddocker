About Service Accounts

Setting up a dedicated service account allows the system to send emails on behalf of your organization without relying on individual admin accounts.

Benefits: More reliable email delivery, consistent sender identity, better security practices.

To set up a Google API service account:

  1. Go to the Google Cloud Console and create a project
  2. Enable the Gmail API for your project
  3. Create OAuth 2.0 credentials (client ID and client secret)
  4. Use the OAuth Playground to get a refresh token with the required scopes
  5. Enter the credentials below

Help & Instructions
Required Scopes

The refresh token must be authorized with these scopes:

  • https://www.googleapis.com/auth/gmail.send
Steps to Get a Refresh Token
  1. Go to OAuth 2.0 Playground
  2. Click the gear icon in the upper right corner
  3. Check "Use your own OAuth credentials"
  4. Enter your Client ID and Client Secret
  5. Select the Gmail API v1 scope
  6. Click "Authorize APIs" and follow the prompts
  7. Click "Exchange authorization code for tokens"
  8. Copy the refresh token
