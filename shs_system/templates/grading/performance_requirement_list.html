About Performance Requirements

Performance requirements determine:

  • Minimum Average Score: The minimum average score a student needs to be promoted.
  • Maximum Failed Subjects: The maximum number of subjects a student can fail and still be promoted.
  • Calculation Method: How student averages are calculated (simple or weighted average).

Only one set of requirements can be active at a time.
